# Power BI Dashboard

This folder contains the Power BI dashboard and related files developed for the AdventureWorks Sales Analysis project.

## Contents

- AdventureWorks Dashboard.pbix
- Dashboard Report.pdf
- Dashboard Screenshots

## Dashboard Features

- Executive KPI Dashboard
- Sales Analysis
- Customer Analysis
- Product Analysis
- Territory Analysis
- Return Analysis
- Interactive Slicers
- Drill-through Navigation
- Dynamic Visualizations

## Tools Used

- Power BI Desktop
- DAX
- Power Query
- MySQL
