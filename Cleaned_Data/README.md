# Cleaned Data

This folder contains the cleaned and validated datasets used for building the SQL data warehouse.

## Description

The datasets in this folder were checked for:
- Missing values
- Duplicate records
- Data type consistency
- Invalid values
- Data quality issues

These cleaned datasets were used to populate the dimension and fact tables in the data warehouse.

## Note

The original source files are available in the `Raw_Data` folder. This folder contains only the processed datasets used during the ETL workflow.
