# Raw Data

This folder contains the original AdventureWorks dataset used for this project.

## Description

The files in this folder are the raw CSV datasets imported into MySQL to build the data warehouse. These datasets were used to create dimension tables and fact tables following the Star Schema design.

## Files Included

- customer.csv
- date.csv
- product.csv
- productcategory.csv
- productsubcategory.csv
- territory.csv
- sales.csv
- returns.csv

## Note

These files are the original source data and were not modified. Data validation and exploration were performed separately using Python before loading the data into the SQL data warehouse.
